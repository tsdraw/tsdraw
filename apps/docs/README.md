# tsdraw docs

Welcome to the [tsdraw docs](https://tsdraw.com/docs) repository. Right now most pages are an AI-generated placeholder (they're probably accurate enough but don't be sure of it). When I have time, I will write the documentation myself. The pages with ✅ next to them are human-written.

Our documentation site is built on [Mintlify](https://mintlify.com) and uses MDX files for the content.

## Development

To build the docs, you need to have the Mintlify CLI installed. You can install it by running the following command:

```bash
npm i -g mint
```

Then, you can run the following command to preview the docs locally:

```bash
mint dev
```
